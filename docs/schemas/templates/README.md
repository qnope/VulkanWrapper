# Schema Templates

This directory contains reusable Mermaid diagram templates for VulkanWrapper documentation.

## Usage

Copy the relevant template and customize it for your specific documentation needs.
All diagrams use Mermaid syntax and render directly in Docusaurus.

## Available Templates

| Template | Use Case |
|----------|----------|
| `architecture.md` | System/module architecture |
| `pipeline.md` | Render pipeline flows |
| `lifecycle.md` | Resource creation/destruction |
| `class-hierarchy.md` | Type relationships |
| `sequence.md` | Temporal interactions |

## Template Index

### 1. Architecture Diagrams

Show high-level component relationships.

### 2. Pipeline Flow Diagrams

Illustrate data flow through rendering stages.

### 3. Resource Lifecycle Diagrams

Show object creation, usage, and cleanup patterns.

### 4. Class Hierarchy Diagrams

Document type hierarchies and inheritance.

### 5. Sequence Diagrams

Show temporal interactions between components.
